Color Families

.bg-{color}, .tx-{color}, .bd-{color}, .ol-{color} — Semantic and utility colors for backgrounds, text, borders, and outlines.

Note: When using color classes, you must also include the base class: use .bg .bg-primary (not just .bg-primary), .tx .tx-primary, and .bd .bd-primary.
