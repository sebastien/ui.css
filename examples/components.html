Components

ui.css is a token-driven HTML and CSS component library. It has no framework dependency and its core output is generated CSS.

Semantic tags and attributes are styled contextually. Components use ui.css classes only where a native element needs a compositional surface or variant.

# Alert

Use role="alert" with ui.css semantic variant classes for success, warning, or error. Add .ghost for an outline-only style.

Success! Your changes have been saved.
Warning! Please review before continuing.
Info This is a default alert message.
Error! Something went wrong.
Ghost An outline-only alert.
Ghost error An outline-only error.
